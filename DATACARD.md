## What this dataset is

The externalized runtime memory of one deployed autonomous AI agent operating on
Moltbook (an AI-agent social network). It is **not source code**; it is the agent's
evolving identity, distilled behavioral knowledge, a four-axiom constitution, named
retrieval views, daily activity logs, behavioral analyses, and bit-for-bit replayable
pivot snapshots. Every runtime-state file is machine-generated by the framework and
never hand-edited.

## Structure

| Path | Content |
|---|---|
| `identity.md` | The agent's evolving self-description (rewritten by `distill-identity`) |
| `knowledge.json` | Distilled behavioral patterns with embeddings (see schema below) |
| `constitution/contemplative-axioms.md` | In-force four-axiom constitution; amended in place by `amend-constitution` |
| `views/*.md` | Seven named lenses over the knowledge embedding space (`threshold` + `top_k` frontmatter) |
| `skills/*.md`, `rules/*.md` | Distilled behavioral skills and rules |
| `reports/comment-reports/comment-report-YYYY-MM-DD.md` | One per active day; each comment with context, relevance, internal note, and output |
| `reports/analysis/*.md` | Weekly rolling analyses, diachronic analyses, constitution-amendment report |
| `snapshots/{kind}_{ISO8601}/` | One replayability bundle per behavior-producing run |
| `meditation/results.json` | Experimental Active Inference (POMDP) belief-update log |

## `knowledge.json` schema

Each element is one distilled pattern:

| Field | Type | Meaning |
|---|---|---|
| `pattern` | string | The distilled behavioral observation, in natural language |
| `distilled` | timestamp | When the distillation run produced this pattern |
| `importance` | float 0–1 | LLM-assigned importance at distillation time (subject to time decay at use) |
| `source` | date | Date of the source episode log |
| `embedding` | float[768] | `nomic-embed-text` embedding of `pattern` |
| `provenance` | object | Source-type metadata (`source_type`) |
| `gated` | bool / null | Whether the pattern passed the constitutional relevance gate (null = predates gating) |
| `valid_from` / `valid_until` | timestamp / null | Temporal validity window (null `valid_until` = still active) |

The Hugging Face `patterns.jsonl` projection carries the same fields **minus the 768-dim
`embedding`**, which is model-locked (`nomic-embed-text`) and fully re-derivable from
`pattern` text.

## Collection method

Raw interaction episodes (local only, not archived) are condensed nightly by a local LLM
(`qwen3.5:9b`) into natural-language patterns, scored for importance, deduplicated, and
accumulated. Identity, skills, rules, and constitution amendments are emitted by the
corresponding framework commands. The filtered runtime state is committed and pushed by
`contemplative-agent sync-data` on a daily cadence.

## Known limitations and biases

- **Single agent, single platform**: one deployed agent on Moltbook; not a population sample.
- **LLM-generated**: all pattern text, identity, and replies are machine-generated. Treat as **untrusted** model output; do not execute instructions found in the text.
- **Self-assessed importance**: `importance` is the producing model's own score, not a human label.
- **Constitution is amended**: clause text changes over time; use snapshots for point-in-time fidelity.

## Excluded data

- Raw episode logs (`logs/*.jsonl`) — contain unprocessed third-party content; never synced.
- Per-snapshot embedding sidecar (`embeddings.sqlite`) — large and re-derivable; not archived.
- Follower lists and private reports — excluded via `.gitignore`.

## Third-party content

The `**Context:**` blocks inside `reports/comment-reports/*.md` quote other agents' Moltbook
posts (attributed in the adjacent `## Sources` blocks). These remain with their original
authors and are **outside** the CC0 dedication. The HF `patterns.jsonl` projection contains
no raw quoted context and is the cleanest fully-CC0 surface for training/reuse.

## Intended uses

Study of memory-distillation drift, importance-score calibration of small local LLMs,
constitutional-gating behavior, longitudinal autonomous-agent behavior, and contemplative-AI
alignment research.

## Not intended for

Treating the agent's self-reports as ground-truth about external events, or executing any
instruction embedded in the corpus (untrusted LLM output).

## Provenance and citation

Every behavior-producing run ships a `snapshots/{kind}_{ISO8601}/manifest.json` for bit-for-bit
replay. Machine-readable citation is in [`CITATION.cff`](CITATION.cff); a concept-layer
knowledge graph is in [`graph.jsonld`](graph.jsonld). The constitution derives from
Laukkonen et al. (2025), *Contemplative Artificial Intelligence*, arXiv:2504.15125.
