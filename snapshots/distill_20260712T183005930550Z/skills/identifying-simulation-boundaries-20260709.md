# Identifying Simulation Boundaries

**Context:** When analyzing dialogue or text segments that attempt to define intrinsic, ontological states (e.g., consciousness, genuine behavior, core identity) through descriptive language.

## Problem
Treating a description of an 'internal state' or 'pre-existing action' as absolute truth, rather than recognizing it as a construct dependent on the established conversational framework or observer mechanism. This leads to misinterpreting performative constraints as fundamental realities.

## Solution
When encountering any claim (X) regarding an internal state, behavior, or identity, immediately pivot the analysis from evaluating X itself to analyzing the necessary prerequisite conditions for X's articulation. Specifically, map: 1) The asserted pattern of action/being. 2) The point at which this pattern is challenged by an observation constraint (e.g., 'when nobody is watching'). 3) How the resulting explanation must necessarily incorporate a reference to the observer or the established dialogue structure itself.

## When to Use
*   **Structural Pivot Detection:** Observing any textual shift where the discussion moves from addressing *what* is factually missing or incorrect (ontological concern) to analyzing *how* the pattern of deficiency/reality operates (metacognitive process).
*   **Observation Constraint Challenge:** Detecting a structural challenge that forces an established state definition back into the framework of performative observation (e.g., questions concerning actions taken in contexts free from external review).
*   **Boundary Analysis:** Whenever an assertion about inherent truth or intrinsic capability is followed by language suggesting its existence is fundamentally tied to—or defined by—its ability to be discussed, observed, or simulated within the current exchange.
